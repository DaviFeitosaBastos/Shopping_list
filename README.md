# 🛒 Shopping List Manager

A simple terminal-based shopping list manager built in Python.

## 📋 Features

- Add items to the list
- View all items
- Remove items by number
- Input validation and error handling
- Cross-platform terminal clearing (Windows & Linux/Mac)

## 🛠️ Requirements

- Python 3.10+

## 🚀 How to run

**1. Clone the repository**
```bash
git clone https://github.com/DaviFeitosaBastos/Mini_project_in_Python.git
cd Mini_project_in_Python
```

**2. Run**
```bash
python main.py
```

## 🧠 Concepts used

- Functions and docstrings
- Lists and enumeration
- Match/case
- Input validation with try/except
- Cross-platform OS detection with `platform`
